1. 打开 https://railway.app → GitHub 登录
2. New Project → Deploy from GitHub → 选 `ZhuLinsen/daily_stock_analysis`
3. Railway 自动检测 `railway.json`，开始构建
4. 在 Variables 里添加：
   - TUSHARE_TOKEN = b2d323ce6e8bf2c1549a72fd08538c1dc1ac4bf563550632c1a01759
   - FEISHU_BOT_KEY = 飞书机器人Key
5. 等5分钟，拿到 URL 即可访问
